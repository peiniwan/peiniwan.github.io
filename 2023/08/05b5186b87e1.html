Vim

[[vim 学习]]

学习vim：vimtutor


嘴巴（kou）在上，老二在下（jiji）
记忆：钱在后，^朝上在前

使用 Ctrl + ] 快捷键，这将会跳转到该方法的定义。
如果你需要返回之前的位置，可以使用 Ctrl + t 或者 Ctrl + o 快捷键进行跳回。

基本模式

单词间定位

  • w（向前跳到单词词首）

  • e（向前跳到单词词尾）

  • b（向后跳到单词词首）

  • 输入 2w 使光标向前移动两个单词。

  • 输入 3e 使光标向前移动到第三个单词的末尾。

  • 输入 0 (数字零) 移动光标到行首。

  • d2w删除2个单词

  • ==按“ea” （记为“End Append”，要在单词前面加东西，就按“bi” （记为 “Begin Insert”）==

    、. 跳到最后一次修改的地方

块定位(shift)

块级操作

  • {（跳到上一个代码块）
  • }（跳到下一个代码块）
  • ==输入shift+ % 可以查找配对的括号 )、]、}==

页面定位

  • gg（定位到页面的第一行）

  • G（定位到页面的最后一行）

  • H（定位到当前页面的顶部，High首字母大写）

  • M（定位到当前页面的中部，Middle首字母大写）

  • L（定位到当前页面的底部，Low首字母大写）

  • nG或ngg（定位到指定行，n表示行数）

屏幕滚动

向上的三角箭头

  • Ctrl + f （向下滚动一屏内容，Foward首字母小写）
  • Ctrl + b （向上滚动一屏内容，Backward首字母小写）
  • Ctrl + e（向下滚动一行内容）
  • Ctrl + y（向上滚动一行内容）
  • zz 改变窗口的整体位置，将当前光标所在处 移动到窗口中部

行内定位

shift

  • ^（定位到行首，忽略空格）
  • $（定位到行尾）
  • 0（定位到行首）
  • j$ 将光标移动到下一行末尾

编辑模式

插入

  • i（在光标处插入，insert首字母小写）

  • I（==在行首插入，insert首字母大写==） // 可以多用

  • a（在光标处后一个字符插入，append首字母小写）

  • A（==在行尾插入，append首字母大写==）

  • o（在上一行插入）

  • O（在下一行插入）

  • cc/S（清除当前行并进入插入模式）

  • s（清除当前字符并进入插入模式）

查找

  • /pattern（正向查找）
  • ?pattern（反向查找）
  • n（查找下一个）
  • N（查找上一个）
  • *（当光标定位在某个单词时，查找下一个该单词）
  • （当光标定位在某个单词时，查找上一个该单词）
  • f; 跳转到字符 ; 处，这里其实就是相当于行位

剪切、复制、粘贴

  • dd（剪切当前行的内容）2dd 删除两行
  • dw删除单词
  • d$ 或 d1g从光标处删至当前行尾部
  • dG删除到文件结尾
  • yy/Y（复制当前行的内容）
  • ye,yw 复制单词
  • p（粘贴到光标后）
  • ==P（粘贴到光标前）==
  • 调换上下两行，直接就记成“ddp”
  • c (范围修改)，进入插入模式
  • ce 修改到词尾、c$
  • ciw 删除当前处的词，并进入 插入模式 其他也可以和==iw==结合
  • x 删除字符

组合操作

v后执行$选中最后一个字符
组合操作可以看作是op+i/a+scope公式的应用，解释一下这个公式，op就是我们前面提到的插入（c）、剪切（d）、复制（y）以及还未提到的选择（v），i表示scope范围内，a表示包含scope标签，scope就是操作的范围了。这样讲比较抽象，下面举例几个常用组合。

选择文本
以下如果将i换成a，则会将符号也包含进去

  • vi(（选中小括号内的内容）
  • vi{（选中大括号内的内容）
  • vi”（选中双引号内的内容）
  • vi’（选中单引号内的内容）
  • vi<（选中尖括号内的内容）

同理，将选择操作（v）替换成插入（c）、剪切（d）或者复制（y），也能实现类似的效果。

撤销

  • u 撤销
  • 大写的 U ，恢复到该行的原始状态

其他

  • gd 方法的引用处
  • ZZ 保存和退出
  • gm 将光标移动到上一个方法的声明处

Vimium

浏览当前页面

  • j 向下滑动
  • k 向上滑动
  • gg 快速到达页面顶部
  • G 快速到达页面底部
  • r 重新载入页面
  • d 向下翻动半页
  • u 向上翻动半页
  • yy 复制当前页面链接至剪贴板
  • H 后退
  • L 前进

打开新的页面

  • o 搜索网址，书签，或历史记录，在当前页面打开
  • O 搜索网址，书签，或历史记录，在新的页面打开
  • b 搜索书签，在当前页面打开
  • B 搜索书签，在新的页面打开

查找

  • / 进入查找模式，输入关键字查找，ESC退出
  • n 切换到下一个匹配
  • N 切换到上一个匹配
  • gi 跳转到当前文本框/搜索框
  • ?:查看所有快捷键

切换tab

  • J 切换到左边tab
  • K 切换到右边tab shift
  • shift 远的是切换前一页后一页（H,L）,记忆：不常用的离得远
  • g0 切换到第一个tab
  • g$ 切换到最后一个tab
  • ^ 切换到刚才的tab
  • t 创建一个新的页面
  • x 关闭当前页面
  • X 恢复刚才关闭的页面
